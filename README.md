# Installation

## Python

```
 $ ln -s ~/bin/dotfiles/python/pythonrc .pythonrc

 $ export PYTHONSTARTUP=~/.pythonrc
```
## Ruby 

```
 $ ln -s ~/bin/dotfiles/ruby/irbrc .irbrc
 $ gem install bond

```
## Zsh
```
 $ ln -s ~/bin/dotfiles/zsh/zshrc .zshrc
```
